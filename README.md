# Calculator
> Calculator in Java with Spring and Thymeleaf.

## General info
Simple application based on Spring Boot and Thymeleaf. It helps with basic Math operations.

## Technologies
* Maven 4.0.0
* Spring Boot 2.1.0
* Thymeleaf 3.0.11

## How to setup
1. Clone this repository <br />
2. Get the dependencies <br />
3. Start project <br />

## Calculator features
* Addition
* Subtraction
* Multiplication
* Division
